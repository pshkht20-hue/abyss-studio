# Premium Site Starter

Шаблон AI-навыков для премиальных сайтов (Next.js / React / GSAP / Framer / Three.js).

## Для агента

1. Роутер скиллов: `.cursor/skills/skill-library/SKILL.md`
2. Анимации: `.cursor/skills/ofm-animation-mastery/SKILL.md` (адаптируй под текущий бренд)
3. Профиль: `.cursor/ecc-site-profile.json`
4. User-level GSAP: `~/.agents/skills/gsap-*`

## Новый сайт в этой папке

После `create-next-app` — stack: Next.js 16+, React 19, Tailwind 4, Framer Motion, GSAP (по необходимости).

Приоритет: **конверсия и контент** > декоративные анимации. Mobile ≤767px lite, `prefers-reduced-motion` обязателен.
